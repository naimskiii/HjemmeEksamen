Github collaboratin

Github are amazing tool for developers who work together on code projects.

Git is allows developert to track changes and organize them in historical versions, while Githu provides a platform for storing,sharing and collaborting on code.

What are the the benifts of Github

  • Using Github makes it easier to get good documentation.
  • Github is a repository, meaning it allows your work to get out to the public
  • You wil always have acsses to your complete history
  • We can also work with unlimited collaboration on small and big projects.!

how did the group procject go?

Collaboration within repositories on GitHub went good and has been an excellent experience. The platform has tausght us how to work as a group.Github made it easy to work together on the project, providing a semless environment for group project. We also had several meeting to discuss the progress and plan the next step, and also being able to se what the other team mebers are doing on github.

What was the challanges with using github

Throughout the process, there were a couple challenges. We encountered a few errors, especially with pushing and pulling changes to and from the repository. The conflicts arose mostly when multiple people made changes to the same part of the project, but these were resolved through clear communication and team effort. Another thing was that sometimes we deleted each other's part by accident and we managed to add and fix this with good communication and collaboration throughout the process. GitHub's version control system allowed us to track the changes, which made it easier to identify and fix any issues.
